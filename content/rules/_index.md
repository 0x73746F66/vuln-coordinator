description: "Write WAF, IPS, detection, and SIEM rules — then reference them in a VEX statement."
weight: 20
---

When a vulnerability is mitigated by a control — a WAF rule, an IPS signature, a YARA detection, a Sigma alert — the mitigation needs to be documented. These guides explain how to write the rule for each platform and how to reference it in an OpenVEX statement as evidence that the risk is managed.
